# Apple-Style Calculator (Java)

A Java Swing calculator app designed to mimic Apple’s Calculator app. Built with a focus on clean custom UI theme, accurate logic, and button interactions. Program uses object-oriented design and event-driven programming to manage app state and arithmetic logic.

## How to Run
1. Make sure Java is installed (Java 8 or later)
2. Clone the Repository:
   ```bash
   git clone https://github.com/your-username/apple-style-calculator-java.git
3. Run the Calculator.Java file:
   ```bash
   javac Calculator.java
   java Calculator

## Features
- Apple-inspired calculator UI using Java Swing

- Custom color palette and button styling/sizing

- Supports standard arithmetic operations (+, -, ×, ÷)

- Square root, percentage, and sign toggle functions

- Decimal inputs

- Clear (AC) function

- Button-based event handling

- Custom logic to format outputs by removing unnecessary decimal zeros

- Defensive user input handling (e.g. preventing multiple decimal points)

## Technologies Used
- Java

- Java Swing (JFrame, JPanel, JButton, JLabel)

- Object-Oriented Programming (OOP)

- Event-driven programming (ActionListener)

- Java Abstract Window Toolkit (AWT) for layout, colors, and fonts
